# Personal Portfolio Website

## Overview

This project is a Personal Portfolio Website developed using HTML, CSS, and JavaScript. It is designed to showcase my profile, education, skills, and contact information in a modern and responsive way. The portfolio provides a professional online presence and highlights my journey as a Computer Science Engineering student and aspiring web developer.

## Features

* Modern and attractive user interface
* Fully responsive design
* Animated typing effect
* Glassmorphism design elements
* Smooth scrolling navigation
* Skills showcase section
* Education section
* Contact information section
* Professional profile image display
* Interactive hover effects and animations

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome Icons

## Sections Included

### Home

Displays a welcome message, profile image, introduction, and navigation menu.

### About

Provides information about my background, interests, and career goals.

### Education

Shows my academic details and current degree information.

### Skills

Highlights technical skills such as HTML, CSS, JavaScript, Python, C Programming, and GitHub.

### Contact

Provides email and GitHub information for communication and networking.

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Ensure all files are placed correctly.
4. Open `index.html` in any web browser.
